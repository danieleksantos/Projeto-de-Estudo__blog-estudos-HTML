h1 a h6

As tags h1, h2, h3, h4, h5 e h6 são elementos HTML utilizados para definir títulos e subtítulos em uma página da web, organizando o conteúdo hierarquicamente.
h1: Representa o título principal da página, geralmente usado apenas uma vez para destacar o tema mais importante.
h2: Subtítulos principais que dividem o conteúdo em seções relevantes.
h3: Subtítulos menores que detalham as seções definidas pelos h2.
h4, h5 e h6: Usados para criar subdivisões ainda mais específicas, com importância decrescente.
Essa hierarquia não só melhora a organização visual, mas também é essencial para a acessibilidade e para otimização de mecanismos de busca (SEO), ajudando os motores de busca a entenderem melhor o conteúdo da página.

p

voltar

A tag "p" é usada no HTML para definir um parágrafo de texto.
Ela organiza o conteúdo em blocos distintos, facilitando a leitura e a estruturação de informações em uma página.
Por padrão, os navegadores aplicam um pequeno espaço antes e depois de cada parágrafo, garantindo separação visual entre os blocos de texto.
A tag "p" é fundamental para conteúdos descritivos, explicações ou qualquer tipo de texto corrido que precise ser apresentado de forma clara e legível.

mark

voltar

A tag "mark" é usada no HTML para destacar trechos de texto, geralmente simulando o efeito de marca-texto.
O conteúdo dentro dessa tag aparece com um fundo colorido (normalmente amarelo) por padrão, chamando a atenção do leitor para informações importantes ou relevantes.
Essa tag é útil para realçar palavras-chave, termos importantes ou resultados de busca em um texto.
Ex: mark

small

voltar

A tag "small" no HTML é usada para indicar um texto de menor importância ou relevância, geralmente exibido em um tamanho de fonte reduzido em relação ao texto ao redor.
Essa tag é comumente utilizada para informações complementares, como notas de rodapé, disclaimers, ou descrições auxiliares.
Visualmente, o texto envolvido pela tag "small" será renderizado em um tamanho menor pelo navegador, mas o estilo pode ser customizado via CSS.

Ex: Este produto contém apenas 50 calorias por porção.

i

voltar

A tag "i" no HTML é usada para exibir texto em itálico, geralmente com o objetivo de dar ênfase a palavras, frases ou expressar um significado específico, como termos técnicos, estrangeirismos, ou pensamentos.
Apesar de seu impacto visual, a tag "i" não transmite automaticamente significado semântico, por isso, em casos onde o itálico representa ênfase ou importância, a tag "em" é mais recomendada.

Ex: A palavra déjà vu é de origem francesa.

u

voltar

A tag "u" no HTML é usada para sublinhar o texto dentro dela, criando um destaque visual.
Seu uso atual é mais voltado para estilização de textos que possuem um significado específico, como nomes próprios em um manuscrito ou termos técnicos em documentos legais.
No entanto, para sublinhar texto por razões puramente estilísticas, é recomendado usar CSS em vez da tag "u" para separar a aparência do significado semântico.

Ex: Por favor, assine na linha marcada com seu nome completo

strong

voltar

A tag "strong" no HTML é usada para indicar um texto de grande importância ou relevância, atribuindo a ele um significado semântico forte.
Por padrão, os navegadores exibem o texto envolvido por "strong" em negrito para transmitir visualmente sua ênfase.
Diferentemente da tag "b", que é puramente estilística, "strong" sugere que o conteúdo tem maior destaque no contexto da página, o que também melhora a acessibilidade em leitores de tela.

Ex: É obrigatório apresentar um documento com foto para entrar no evento.

ol

voltar

A tag "ol" no HTML é usada para criar uma lista ordenada, onde os itens são exibidos em sequência numérica ou em outro tipo de ordenação, como letras ou algarismos romanos.
Cada item da lista é representado por uma tag "li" (list item). As listas ordenadas são úteis para exibir passos sequenciais, classificações ou informações hierárquicas.

Ex:

  1. Preaqueça o forno a 180°C.
  2. Misture os ingredientes secos.
  3. Adicione os ingredientes líquidos e misture bem.
  4. Despeje a massa na forma e asse por 30 minutos.

ul

voltar

A tag "ul" no HTML é usada para criar uma lista não ordenada, onde os itens são exibidos com marcadores (geralmente pontos ou outros símbolos) em vez de uma sequência numérica.
Cada item da lista é representado por uma tag "li" (list item). As listas não ordenadas são ideais para apresentar informações em grupos, sem a necessidade de uma ordem específica.

Ex:

  • Tomate
  • Alface
  • Cenoura

li

voltar

A tag "li" no HTML é usada para definir um item dentro de uma lista, seja ela ordenada (ol) ou não ordenada (ul).
Cada "li" representa um único item dentro da lista e é envolvido pelas tags "ul" ou "ol".
Essa tag ajuda a estruturar informações de forma clara e organizada, tornando o conteúdo mais legível e fácil de navegar.

Ex:

  • Leite
  • Pão
  • Queijo

a

voltar

A tag "a" no HTML é usada para criar links, permitindo a navegação para outras páginas, seções, ou recursos externos.
O atributo mais comum é o href, que especifica o destino do link.
Ao clicar no texto, o usuário será redirecionado para o endereço definido no atributo href.
A tag também pode incluir atributos como target="_blank" para abrir links em novas abas.

Ex: https://www.dio.me/

hr

voltar

A tag "hr" no HTML é usada para criar uma linha horizontal que geralmente indica uma mudança de tema ou uma separação entre seções de conteúdo.
Essa linha tem um propósito semântico, marcando uma pausa ou divisão lógica no fluxo do texto. Por ser uma tag autossuficiente, ela não precisa de uma tag de fechamento.

sub

voltar

A tag "sub" no HTML é usada para exibir texto em subscrito, ou seja, em uma posição inferior à linha base do texto ao redor.
Ela é comumente empregada em fórmulas químicas, expressões matemáticas ou anotações técnicas.

Ex: A fórmula da água é H2O.

sup

voltar

A tag "sup" no HTML é usada para exibir texto em sobrescrito, ou seja, em uma posição elevada acima da linha base do texto ao redor.
Ela é frequentemente utilizada em expressões matemáticas, fórmulas químicas ou referências, como notas de rodapé.

Ex: A área de um quadrado é dada por lado2.

blockquote

voltar

A tag "blockquote" no HTML é usada para marcar citações em bloco, indicando que o texto é uma citação longa de outra fonte.
Ela costuma ser exibida com um recuo ou estilização diferenciada pelos navegadores, destacando visualmente o conteúdo citado.

Ex:

"O sucesso é ir de fracasso em fracasso sem perder o entusiasmo." - Winston Churchill

font

voltar

A tag "font" no HTML é usada para alterar o estilo do texto, incluindo atributos como cor (color), tamanho (size) e tipo de fonte (face).
No entanto, essa tag é considerada obsoleta no HTML5, e seu uso não é mais recomendado. Para estilizar texto, é preferível usar CSS, que oferece maior flexibilidade e separa a apresentação do conteúdo.

Ex: Texto estilizado

del

voltar

A tag "del" no HTML é usada para indicar texto que foi removido ou descartado, como em edições ou revisões de conteúdo.
Os navegadores geralmente exibem o texto com um traço (riscado) para indicar a exclusão.
Essa tag possui valor semântico, ajudando a destacar alterações no texto, especialmente em documentos colaborativos ou históricos.

Ex: O preço original era R$ 100,00, agora está por R$ 80,00!

abbr

voltar

A tag "abbr" no HTML é usada para marcar abreviações ou siglas, permitindo que os navegadores forneçam informações adicionais sobre o significado do texto.
Geralmente, um atributo title é adicionado para especificar o que a abreviação representa, melhorando a acessibilidade e a compreensão do conteúdo.

Ex: O HTML é uma linguagem de marcação para a web.

img

voltar

A tag "img"" em HTML é usada para inserir imagens em uma página web. Ela é um elemento vazio, ou seja, não tem conteúdo entre as tags de abertura e fechamento, e seus atributos principais são:

  • src: especifica o caminho da imagem (URL ou arquivo local).

  • alt: fornece um texto alternativo para ser exibido caso a imagem não carregue ou para leitores de tela, melhorando a acessibilidade.

  • width e height: definem as dimensões da imagem (em pixels ou porcentagem).

  • alt (obrigatório): fornece uma descrição textual da imagem para usuários que usam leitores de tela ou quando a imagem não carrega. Deve ser informativo, mas pode ser vazio (alt="") se a imagem for puramente decorativa.

  • role="presentation" ou aria-hidden="true": usado em imagens decorativas para indicar que elas devem ser ignoradas por leitores de tela.

audio

voltar

A tag "audio" em HTML é usada para incorporar arquivos de áudio em uma página web.
Ela permite reproduzir sons, como músicas ou efeitos, diretamente no navegador.
Diferente da tag "img", "audio" pode ter um conteúdo entre suas tags de abertura e fechamento, como texto ou elementos de fallback para navegadores que não suportam áudio.
Atributos principais:

  • src: define o caminho do arquivo de áudio.

  • controls: exibe controles padrão (play, pause, volume) para o usuário.

  • autoplay: inicia a reprodução automaticamente.

  • loop: faz o áudio se repetir automaticamente.

  • muted: inicia o áudio no modo mudo.

  • aria-label ou aria-labelledby: fornece uma descrição do conteúdo do áudio, especialmente se os controles padrão não forem exibidos.

  • Legendas e transcrições: para conteúdo significativo, deve-se oferecer uma transcrição do áudio, seja em texto diretamente na página ou em um link.

video

voltar

A tag "video" em HTML é usada para incorporar vídeos em uma página web.
Ela permite exibir conteúdos multimídia diretamente no navegador, oferecendo recursos visuais e interativos.
Assim como a tag "audio", "video" pode conter elementos "source" para especificar várias fontes de vídeo e texto de fallback para navegadores que não suportam vídeos.
Atributos principais:

  • src: define o caminho do arquivo de vídeo.

  • controls: exibe controles padrão como play, pause e volume.

  • autoplay: inicia a reprodução automaticamente.

  • loop: faz o vídeo se repetir continuamente.

  • muted: inicia o vídeo sem som.

  • poster: define uma imagem exibida enquanto o vídeo não é reproduzido.

  • aria-label ou aria-labelledby: descreve o conteúdo do vídeo de forma acessível.

  • alt no atributo poster: se o vídeo usa um pôster (poster), o atributo alt deve ser fornecido para descrever a imagem.

  • Descrição de áudio: adicionar uma faixa de áudio com descrições narradas para vídeos que não são totalmente compreensíveis visualmente.

track

voltar

A tag "track" é usada em HTML para adicionar legendas, descrições ou outros textos sincronizados a conteúdos multimídia, como vídeos e áudios.
Ela deve ser usada dentro de um elemento "video" ou "audio" e suporta diferentes tipos de faixas, como legendas (kind="subtitles").
O atributo src indica o arquivo da faixa, enquanto kind, srclang e label especificam o tipo, idioma e rótulo da faixa.

iframe

voltar

A tag "iframe" é usada em HTML para incorporar outro documento HTML dentro de uma página.
Ela cria um quadro embutido que pode exibir páginas da web, vídeos ou outros conteúdos de fontes externas.
